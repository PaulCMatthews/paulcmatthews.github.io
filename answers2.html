  3. Suppose that \(y' = f(y/x) \). Set \( u(x) = y(x)/x\), so \(y=x u, \ y' = u+ x u'\). Then \(u \) obeys the equation $$ u+xu' = f(u)$$. Hence $$u' =\frac{f(u)-u}{x},$$ which is separable. For $$ y' = \frac{ x^2 + y^2}{xy}= \frac{1+y^2/x^2}{y/x}, $$ $$x u' =\frac{1 + u^2 }{u}-u= \frac{1}{u}$$ which leads to $$y= \pm x \sqrt{2\ln x + c}$$ For the initial condition \(y(1)=1\), \(c=1\) and we take the + sign.
